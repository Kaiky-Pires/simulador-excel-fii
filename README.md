# Simulador de Investimentos - Fundos Imobiliários

Projeto prático desenvolvido para o desafio da **DIO**, focado na criação de uma ferramenta de controle e simulação de FIIs.

## Desenvolvimento Mobile
Este projeto foi realizado integralmente através do **Microsoft Excel para Mobile**, aplicando técnicas de:
- Fórmulas de juros e rendimentos.
- Referências absolutas ($) para automação de cálculos.
- Preenchimento automático de sequências em touchscreen.

## Funcionalidades
- **Entrada de dados:** Valor da cota, dividendo por cota e aporte mensal.
- **Cálculo de Cotas:** Fórmula automática para descobrir quantas cotas o aporte compra.
- **Projeção de 12 Meses:** Tabela dinâmica mostrando o crescimento do patrimônio e dos dividendos mensais.

## Como funciona
A planilha utiliza a lógica de juros simples sobre aportes mensais, onde cada mês soma as novas cotas às anteriores, gerando o efeito de "bola de neve" nos rendimentos.
